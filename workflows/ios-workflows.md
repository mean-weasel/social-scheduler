# iOS Simulator Test Plan

## Core Workflows

1. **Dashboard Overview** - View metrics, upcoming posts, recent drafts
2. **Create & Save Draft** - New post, write content, save as draft
3. **Schedule Single-Platform Post** - Twitter-only scheduled post
4. **Schedule Multi-Platform Post** - Twitter + LinkedIn post
5. **Reddit Cross-Posting** - Multiple subreddits with individual schedules
6. **Edit Existing Post** - Find draft, edit, save changes

## Campaign Management

7. **Create New Campaign** - Name and description
8. **Add Twitter Post to Campaign** - Twitter-only post in campaign
9. **Add LinkedIn Post to Campaign** - LinkedIn-only post in campaign
10. **Add Multi-Platform Post to Campaign** - Twitter + LinkedIn in campaign
11. **Add Reddit Single-Subreddit Post to Campaign** - One subreddit in campaign
12. **Add Reddit Cross-Post to Campaign** - Multiple subreddits in campaign
13. **View Campaign Detail** - Verify all post types display correctly
14. **Edit Campaign** - Update name/description

## List & Management

15. **Posts List & Filtering** - Browse posts, filter by status
16. **Archive & Restore** - Archive post, then restore
17. **Settings** - Theme toggle, notification preferences

## Notes

- Track auto-save behavior while editing
- Check character count limits per platform
- Verify date/time picker functionality
- Test unsaved changes warning on navigation
