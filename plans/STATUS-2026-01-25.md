# Bullhorn Status Report

**Date:** January 25, 2026
**Stage:** Beta / Pre-production

---

## What Is Bullhorn?

A full-stack social media scheduling app for Twitter/X, LinkedIn, and Reddit. Recently rebranded from "Social Scheduler."

---

## Tech Stack

- **Frontend:** Next.js 14, React 18, TypeScript, Tailwind CSS, Zustand
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **Testing:** Playwright E2E (216 tests), Vitest unit tests
- **Integration:** MCP server for Claude automation

---

## Implemented Features

**Core**
- Multi-platform post creation with platform-specific fields
- Post scheduling, editing, archiving, and deletion
- Campaign management (group posts, track status)
- Blog drafts with markdown and image attachments
- Reddit cross-posting to multiple subreddits

**Users**
- Email/password and Google OAuth login
- Email verification with resend
- User profiles (display name, avatar)
- Password reset flow
- Account management (change password, delete account)

**Media & UX**
- Image/video uploads (10MB/100MB limits)
- Real-time character counters per platform
- Search and filter posts by status/platform/campaign
- Auto-save drafts
- Mobile-friendly navigation
